# Sesión de trabajo — 2026-05-19

## Proyecto
**RP Design & Construction** — Sitio web informativo para empresa de construcción en New Jersey, USA.

## Lo que se construyó

### Tecnología
- Next.js 16 + TypeScript + Tailwind CSS v4
- Deploy en Vercel (cuenta gratuita)
- Repo GitHub: https://github.com/jmhert20/rp-design-construction
- URL live: https://rp-design-construction.vercel.app

### Estructura del proyecto
```
rp-design-construction/
├── app/
│   ├── globals.css       — Estilos globales, fuentes Inter + Playfair Display
│   ├── layout.tsx        — Layout raíz con LanguageProvider
│   └── page.tsx          — Página principal (importa todos los componentes)
├── components/
│   ├── Navbar.tsx        — Navbar fija con toggle ES|EN arriba a la derecha
│   ├── Hero.tsx          — Hero split: logo RP + headlines + imagen Unsplash
│   ├── Services.tsx      — 10 servicios con iconos SVG, fondo oscuro
│   ├── Gallery.tsx       — 4 fotos (Driveways, Walkways, Patios, Muros)
│   ├── Contact.tsx       — Info de contacto + CTA card
│   └── Footer.tsx        — Footer con logo y copyright
├── context/
│   └── LanguageContext.tsx — Context React para ES/EN, persiste en localStorage
├── translations/
│   └── index.ts          — Textos completos en español e inglés
└── scripts/
    ├── backup.sh         — Cron backup git cada 3h
    └── save-session.sh   — Snapshot de estado + git commit cada 3h
```
